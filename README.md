## ✅ Features

### 🔝 Header
- TripAdvisor owl logo (SVG)
- Centered search bar with green search button
- Navigation links — Hotels, Flights, Restaurants, Vacation, Sign in
- Hamburger menu for mobile screens

### 🗂️ Category Navigation Bar
- Sticky pill-style nav below the header
- Categories: Hotels, Restaurants, Attractions, Flights, Car Rentals, Cruises, Travel Stories, Forums
- Active state highlight on click

### 🖼️ Hero Section
- Full-screen background image with dark overlay
- Serif display headline using Playfair Display font
- Tabbed search form — Hotels / Flights / Restaurants / Things to Do
- Fields: Destination, Check-in, Check-out, Guests
- Fully responsive search box

### 🔍 Search & Filter Section
- Destination search input
- Three filter dropdowns — Price Range, Min Rating, Type
- Apply Filters button

### 🌍 Featured Destinations
- Asymmetric CSS Grid layout
- Large hero card + 4 smaller destination cards
- Hover zoom effect on images
- Tag badges (Trending, Classic, Popular, Must-see, Luxury)

### 🏨 Popular Hotels
- 4-card responsive grid
- TripAdvisor-style bubble ratings
- Price per night display
- Travellers' Choice & Luxury Pick badges
- Favourite (heart) toggle button
- "View deal" CTA button

### 🍽️ Top-Rated Restaurants
- 4-card responsive grid
- Cuisine tags (Italian, Japanese, Seafood, etc.)
- Open / Closed status indicator
- "Reserve table" CTA button

### 🎡 Things to Do
- Horizontal attraction list cards
- Image + details + price + Book now button

### 💬 Review CTA Banner
- Dark gradient section
- Live stat counters — 1B+ Reviews, 8M+ Listings, 490M+ Visitors
- Write a review & Join free buttons

### 🦶 Footer
- 4-column link grid
- App Store & Google Play badges
- Social media icons (Facebook, X, Instagram, Pinterest, YouTube)
- Copyright and legal links

## 🛠️ Built With

| Technology | Purpose |
|---|---|
| HTML5 | Page structure and semantic layout |
| CSS3 | Styling, Flexbox, Grid, animations |
| CSS Custom Properties | Theming and design tokens |
| CSS Grid | Destination cards and multi-column layouts |
| Flexbox | Header, cards, navbar alignment |
| Media Queries | Responsive design for all screen sizes |
| Font Awesome 6 | Icons throughout the site |
| Google Fonts | Playfair Display (headings) + Inter (body) |

---

## 📱 Responsive Breakpoints

| Breakpoint | Layout |
|---|---|
| Desktop (> 1024px) | Full navbar, 4-column grids, side-by-side layouts |
| Tablet (768px – 1024px) | Hamburger menu, 2-column destination grid |
| Mobile (< 768px) | Single column, stacked hero form, collapsed nav |
| Small Mobile (< 540px) | Full-width cards, stacked filters, centered footer |

## 🚀 Getting Started

### View locally

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/tripadvisor-clone.git
```

2. Navigate into the folder:
```bash
cd tripadvisor-clone
```

3. Open `index.html` in your browser:
```bash
# On Windows
start index.html

# On Mac
open index.html
```

No build steps, no dependencies — just open and go.

---

## 🌍 Deploy on GitHub Pages

1. Go to your repository on GitHub
2. Click **Settings** → **Pages**
3. Under **Branch**, select `main` → click **Save**
4. Your site will be live at:
```
https://YOUR_USERNAME.github.io/tripadvisor-clone/
```

## 📚 What I Learned

- Structuring large HTML pages with semantic elements (`header`, `main`, `section`, `article`, `footer`)
- Using **CSS Grid** for complex asymmetric layouts (destination cards)
- Building a **sticky multi-level navigation** (header + category bar)
- Designing **interactive cards** with hover effects, badges, and rating bubbles
- Making a fully **responsive layout** using media queries without any CSS framework
- Creating a **tabbed search form** with pure HTML and CSS
- Implementing **accessibility** features — `aria-label`, `role`, `focus-visible`
